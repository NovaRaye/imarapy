# imarapy

Python bindings for the [imara-diff](https://github.com/rsms/imara-diff) Rust library, providing efficient and robust text diffing functionality.

[中文文档](README_CN.md)

## Installation

### Via PyPI

```bash
pip install imarapy
```

Please refer to [demo.py](./demo.py) for detailed usage examples.
